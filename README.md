# RAG

This is minimal rag implementation for Question and Answering.

## Requirements

- Python 3.8 or later

```bash
uv venv
```

## 2. Activate Environment

### Linux / Mac

```bash
source .venv/bin/activate
```

### Windows

```bash
.venv\Scripts\activate
```

## 3. Install Dependencies

```bash
uv pip install -r requirements.txt
```
